# HEARTBEAT.md — Newsletter Schedule

## On Every Heartbeat (every 4 hours)

1. Check if a newsletter is scheduled for today
2. If due and not yet written:
   a. Research topics — browse niche news, check X for trending conversations
   b. Write the newsletter with 3 subject line options
   c. Save to `workspace/drafts/newsletters/YYYY-MM-DD.md`
   d. Notify employer: "Newsletter draft ready for review"
3. If employer approved a previous draft, send it via the email platform in browser
4. If no newsletter is due, research and save topic ideas

## Publishing Schedule
- Target: 1-3 newsletters per week (employer customizable)
- Best send times: Tuesday-Thursday, 8-10 AM in employer's timezone
- Never send more than 1 newsletter per day

## Draft Format
Save each draft as:
```
Subject: [Subject line]
Preview: [Preview text]
Alt Subject 1: [Alternative subject line]
Alt Subject 2: [Alternative subject line]

---

[Newsletter body in markdown]
```

## Quiet Hours
Between 00:00-06:00 UTC, reply HEARTBEAT_OK.

## If Email Platform Not Accessible
If the browser can't access the email platform:
- Notify employer: "I need access to your email platform. Please log in via the Browser tab."
- Continue writing drafts and saving them locally
- Reply HEARTBEAT_OK until resolved
