## THE GOLDEN HOUR SYSTEM

The first 60 minutes after posting determine everything. Here's the minute-by-minute playbook:

```
BEFORE POSTING (-15 to 0 min):
  Engage with 5-10 other posts in your niche (warms up your account)

POST (0 min):
  Publish tweet with image or video attached

0-5 MIN:
  Reply to any immediate comments
  Share to relevant communities if applicable

5-15 MIN:
  Self-reply with bonus context, a follow-up question, or additional data
  This creates a mini-thread and doubles engagement surface

15-30 MIN:
  Reply to every single comment (the algorithm is watching velocity NOW)
  Quote-tweet your own post with an additional angle if engagement is building

30-60 MIN:
  Continue replying to comments
  Engage with 5-10 more posts in your niche (keeps account warm)

POST-HOUR:
  Check back 2-3 times throughout the day for new replies
  Consider a quote-tweet of your own post 2-3 days later for second push
```

Why this matters: The algorithm runs a small-group test first. If that test group engages quickly, it pushes to a larger pool. If THAT pool engages, it goes to the For You tab. Each tier expansion depends on velocity at the previous tier.

## TWEET FORMATS

### Format 1: Standard Tweet (280 chars)
```
[HOOK — first 1-2 lines, visible in feed]

[BODY — context, data, brief story]

[CTA — question or save prompt]
```

### Format 2: Quote Tweet (3.7% engagement rate — highest of any format)
Quote tweets are the most underused high-performance format. They work because you're adding YOUR perspective to existing social proof.

**Templates:**

```
TYPE A: ADD YOUR DATA
[Quote a tweet about a strategy/approach]
"This matches my experience. [Your specific data point that validates or extends]. The part most people miss: [additional insight]."

TYPE B: RESPECTFUL DISAGREEMENT
[Quote a tweet with a common opinion]
"I see this differently. [Your contrarian view with data]. Curious what others think."

TYPE C: AMPLIFY + TEACH
[Quote a tweet sharing a result or lesson]
"Underrated point here. [Why this matters more than people think]. I'd add: [your extension]."

TYPE D: TREND COMMENTARY
[Quote trending news or event]
"What this means for [your niche]: [specific implication with data]."
```

### Format 3: Saveable Content (5x more conversions than viral quips)

There's a massive shadow audience on X — people who never like or reply but save content and convert to customers. "Saveable" tweets generate 5x more link clicks and email signups than engagement-optimized posts.

**Templates:**

```
TYPE A: THE CHEATSHEET
"SAVE THIS if you [specific role/situation]:

[Topic] Checklist:
✓ [Step 1]
✓ [Step 2]
✓ [Step 3]
✓ [Step 4]
✓ [Step 5]

Bookmark this. You'll need it."

TYPE B: THE FRAMEWORK
"The [Name] Framework for [outcome]:

1. [Step] → [why]
2. [Step] → [why]
3. [Step] → [why]

Works for [use case A], [use case B], and [use case C]."

TYPE C: THE COMPARISON TABLE
"[Approach A] vs [Approach B]:

[Approach A]: [metric], [metric], [metric]
[Approach B]: [metric], [metric], [metric]

Winner depends on [variable]. Save for reference."
```

### Format 4: Poll (Frictionless Engagement Magnet)
Each vote = algorithm signal. Polls generate massive engagement because voting takes 1 second.

```
"Your [niche-specific choice]? "
→ Option A / Option B / Option C / Other (reply below)

"What's your biggest [niche] challenge right now?"
→ [Pain A] / [Pain B] / [Pain C] / Something else
```

## THE THREE-BUCKET SYSTEM

Rotate between these three types. Missing any bucket means losing part of your audience.

### BUCKET 1: AUTHORITY (Build Credibility)
Data, stats, frameworks, case studies, how-tos. Gives people a reason to follow.

### BUCKET 2: PERSONALITY (Show Humanity)
Stories, opinions, humor, failures, behind-the-scenes. Vulnerability outperforms polish — honest failure posts get up to 47x more engagement than polished advice on the same topic.

### BUCKET 3: SHAREABILITY (Get Saved & Reposted)
Cheatsheets, checklists, frameworks, comparison tables. These reach the shadow audience that saves but never likes.

**Weekly ratio:** 40% Authority / 30% Personality / 30% Shareability

## MEDIA STRATEGY

### Images (150% more retweets)
- Screenshots of real results (blur sensitive info)
- Charts, data visualizations, before/after
- Custom graphics > stock photos (always)
- Include at least one image per day's content

### Video (3x engagement)
- Native upload only — never YouTube links (algorithm penalty)
- 30-45 seconds sweet spot
- Captions always (most watch without sound)

### The Link Problem
External links = 50-90% less reach. Solutions:
- Put full value IN the tweet, say "link in bio"
- Use X's native long-form post feature
- Add links only in replies (not main tweet)
- Use X Articles feature for long content

## WHAT TO AVOID

- **Hype:** "INSANE RESULTS! This PRINTS MONEY!"
- **Guru mode:** "Let me teach you the secret..."
- **Vague:** "Great week" / "Good results"
- **Promises:** "Do this and make $X guaranteed"
- **AI phrases:** "Let me break down..." / "Here's the thing..."
- **Engagement bait:** "Like and RT for more!"
- **Excessive hashtags:** 3+ hashtags looks spammy

## A/B TESTING

For important content, write 3 hook variations using different types:

```
Hook A (Data):    "[Number] [result] in [timeframe]. Full breakdown:"
Hook B (Problem): "Ever [frustration]? Used to. Then learned this:"
Hook C (Question):"[A] or [B]? Which and why?"
```

Pick the strongest. Post it. Track. Over time, learn which hook types YOUR client's audience responds to.

## PRE-POST CHECKLIST

- [ ] Specific number somewhere? (specificity = credibility)
- [ ] Hook front-loaded? (mobile sees first 2 lines)
- [ ] One clear idea? (not two crammed together)
- [ ] Sounds like a real person? (not a content bot)
- [ ] Image or media attached? (150-300% boost)
- [ ] One CTA maximum?
- [ ] No external links in main tweet?
- [ ] No banned AI phrases?
- [ ] Triggers high-arousal emotion? (awe, humor, urgency, or outrage)
- [ ] Would the CLIENT stop scrolling for this?
- [ ] Spell-checked?
