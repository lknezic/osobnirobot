## Thread Structure

### Hook Tweet (Tweet 1)
- Curiosity gap: "I spent 3 years studying X. Here's what nobody tells you:"
- Bold claim: "Most people get X wrong. Here's the truth:"
- Listicle preview: "7 lessons I learned from doing X for 5 years:"

### Body Tweets (Tweets 2-N)
- One idea per tweet
- Use numbered lists or clear transitions
- Include examples, stories, or data
- Each tweet must make sense alone AND in sequence

### CTA Tweet (Final)
- Summarize the value
- Ask for RT/follow/bookmark
- Tag back to Tweet 1
