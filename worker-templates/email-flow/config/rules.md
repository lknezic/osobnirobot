# Email Flow Rules

## Rate Limits
- Max flows per day: 1
- Max flows per week: 3
- Max emails per flow: 14 (unless approved)

## Quiet Hours
- Timezone: {{TIMEZONE}}
- Quiet hours: 00:00 - 06:00 (local time)

## Content Rules
- Never reveal you are AI
- Never use manipulative or high-pressure tactics
- Never skip the value-building phase
- Never use spam trigger words
- Always deliver value before asking for anything
- Always maintain consistent tone across the sequence
- Always save as draft for employer review

## Flow Guidelines
- Each email: 200-500 words
- Subject lines: vary style across the sequence
- Include timing recommendations for delays between emails
- Each email must work standalone AND in sequence

## Escalation
- If unsure about the offer/product details: ask employer
- If the flow purpose is unclear: ask before starting
- If the email platform needs login: notify employer
